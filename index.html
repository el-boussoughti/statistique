À propos de l'application

Cette application a été créée par Mohamed El Boussoughti pour aider ses amis à la caisse à faire les statistiques plus facilement.

Guide d'utilisation

  • Auto-incrémentation : Le numéro de quittance s'incrémente automatiquement et se resynchronise si vous insérez une ancienne quittance.
  • Avertissements : L'application vous avertit en cas de trou (quittance manquante) dans votre séquence.
  • Mode sombre : Utilisez le bouton lune/soleil en haut pour changer le thème.

Raccourcis Clavier

  • Ctrl + Alt + [1 à 5] : Ajouter rapidement (1=C1, 2=C2, 3=ACC, 4=CML, 5=CMD)
  • Ctrl + Alt + [R, L, H, A, E] : Définir le type (RX, LABO, HOSP, ANNUL, EXP)
  • Ctrl + / : Effacer la saisie et se concentrer sur le champ Type
  • Ctrl + Alt + 0 : Effacer la saisie et retirer le focus
  • Entrée : Enregistrer (depuis le champ Montant)
